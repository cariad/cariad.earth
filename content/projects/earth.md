[Earth](https://github.com/cariad/earth) is the Hugo theme I built for [cariad.earth](https://cariad.earth).

## Features

- Responsive and tested on iPhones and iPads
- Respects the system light/dark mode
- Local fonts (no calls to Google Fonts)
- Minimal and local JavaScript
- Tables of content
- OpenGraph / Twitter cards
- Emoji favicon
- Fathom Analytics
- Resizes and converts images to WebP
- Featured images
- Code syntax highlighting with Dracula theme

## Configuration

The full installation and configuration details are online at [cariad/earth/README.md](https://github.com/cariad/earth/blob/main/README.md).

This site's own configuration is documented on GitHub (in fact, if you hadn't noticed already, [this entire site is open-source](https://github.com/cariad/cariad.earth)) at [cariad/cariad.earth/config.yml](https://github.com/cariad/cariad.earth/blob/main/config.yml).

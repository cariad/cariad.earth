Projects

I maintain a number of Python packages on GitHub at github.com/cariad.

Some of my favourites include:

  • cline is a framework for building highly-testable command line applications and asking allows scripts to interactively prompt users for input. I've used both of these packages to build command-line applications for Freyda.
  • greas3 uploads files to Amazon Web Services S3, using checksums to determine if a file is already up-to-date to prevent unnecessary work.
  • lackadaisical identifies slow code during unit testing. I use this for performance testing at Freyda.
  • rolumns manipulates data into rows and columns and recompose transforms lists and dictionaries according to schemas. Both of these packages are used for data manipulation and export at Freyda.
  • slash3 builds and navigates Amazon Web Services S3 URIs, much like the Path class does for building and navigating file system paths.
  • wev runs shell commands with environment variables determined at runtime. I use this tool on an almost-daily basis at Freyda to inject Amazon Web Services CodeArtifact authentication tokens into Pipenv and to perform multi-factor authentication for the AWS command line.

In 2021, I earned a Mars 2020 Contributor badge for my contributions to Boto3, which was using during NASA's Ingenuity helicopter mission on Mars.

In 2022, I released wa11y.co to help Wordle players share screen reader-friendly results on social media. The app was used 31.1k times per month at its peak and earned mentions on The Verge and Slate.
